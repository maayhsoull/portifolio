# Portifólio - May Martins

## Sobre o projeto
Este projeto é um portifólio pessoal desenvolvido com o objetivo de apresentar minhas habilidades, experiências e projetos na área de desenvolvimento.

O site contém seções como:
- Apresentação pessoal
- Habilidades técnicas
- Experiências profissionais
- Projetos desenvolvidos
- Informações de contato

## Template utilizado
Este projeto foi baseado no template "Hyperspace", disponibilizado gratuitamente pelo site HTML5 UP.

Link do template original:
https://html5up.net/hyperspace

## Modificações realizadas
Apesar de ter sido iniciado a partir de um template pronto, diversas alterações foram feitas para personalização, como:

- Alteração completa dos textos e conteúdos
- Inclusão de informações pessoais e profissionais
- Adição de projetos próprios com links
- Modificação das seções (estrutura e organização)
- Ajustes no layout e estilização
- Inserção de links para GitHub e LinkedIn

## Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript
